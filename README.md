# Portfolio 
Source code for my portfolio website hosted at [akxcix.com](https://akxcix.com)
## Running
### Requirements
- docker
### Build Image
``` bash
docker build . -t portfolio
```
### Run Image
``` bash
docker run -p 80:80 portfolio
```
