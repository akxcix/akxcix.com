# [akxcix.com](https://akxcix.com)
Source for my (insanely bloated) website.

## Running Locally
### Dev Mode
``` bash
npm install
npm start
```

### Docker
#### Build Image
``` bash
docker build . -t portfolio
```
#### Run Image
``` bash
docker run -p 80:80 portfolio
```
